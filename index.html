На живых примерах посмотрим, что мождно сделать с помощью переопределение стилей сайта, никакой магии, только пара костылей на чистом CSS!

Это не шутка, всё что вы увидите в рамках данной лекции на самом деле жуткие костыли и ни в коем случае не рекомендуется к использованию в продакшн.

Основы HTML

HTML (от англ. HyperText Markup Language — язык гипертекстовой разметки) — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере. Веб-браузер получает HTML документ от сервера и интерпретируют его в визуальное представление, на данный момент поддерживается встраивание практически любого типа информации, тест, звук, графика и т.д.

HTML страница строится из отдельных элементов, которые называются HTML-тегами, они записываются с использованием угловых скобок, браузеры не отображают их, но используют для интерпретации содержимого страницы.

В основном HTML-теги несут специальное значение, например тег <img> служит для встраивания в HTML-документ изображений, <p> обозначает параграф текста, <iframe> позволяет встраивать другие HTML-документы в текущий, и это далеко не полный перечень HTML-тегов.

<!doctype html>
<html lang="ru">
<head>
    <meta charset="utf-8">
    <title>Мета заголовок страницы</title>
</head>
<body>
    Содержимое страницы
</body>
</html>

Есть и более общие элементы, которые не несут в себе какой-то специальный контекст, например, тег <div> служит для разделения содержимого страницы на семантические блоки и по умолчанию никак не влияет на отображение, пока не будет стилизован.

HTML в отличие от XML не чувствителен к ошибкам, например можно использовать несуществующие теги или вкладывать теги друг в друга, которые этого не предполагают, документ в любом случае отработает и каким-то образом отрисуется браузером, но это не значит что так можно делать, спецификация может измениться в любой момент, и есть большой шанс получить непредвиденное поведение, поэтому лучше так не делать.

<some-non-existent-tag>Пример несуществующего тега.</some-non-existent-tag>

    Структура HTML-документа представляет из себя дерево, его называют DOM (от англ. Document Object Model), HTML-теги вкладываются друг в друга для получения нужного результата. Наравне с основной структурой HTML-документа существует вспомогательная так называемый теневой DOM (от англ. Shadow DOM), в основном он служит для отрисовки внутренностей HTML-тегов, например, <select> или скроллбар окна браузера, как правило на такие элементы нельзя повлиять извне, или доступ к ним будет сильно ограничен.

    По умолчанию блоки страницы выстраиваются сверху вниз, независимо от своего семантического значения, конечно если это не inline элементы, которые выстраиваются в строку, например ссылку можно разместить внутри текста это поведение HTML-тега <a> по умолчанию, но его можно изменить с помощью языка стилей CSS.

    Основы CSS

    CSS (от англ. Cascading Style Sheets - каскадные таблицы стилей) - это специальный язык разметки стилей, которые могут применяться к HTML-документу, был создан для отделения логики отвечающей за внешний вид от разметки страницы. Он позволяет изменить шрифт, цвет, размер или позицию элементов на странице, а также даёт возможность изменять эти свойства плавно, получая эффект анимирования.
